AI Chat Application

Using Llama, Tkinter, and psutil to build a modern, interactive AI assistant.

About the Project

This AI Chat Application leverages the Llama model, a local language model, integrated with Tkinter for a sleek GUI. The app interacts with the user, responds intelligently, and can access system health data like CPU and RAM usage. The focus is on concise, human-like AI interaction in a user-friendly interface.

Key Features

  • User-friendly chat interface using Tkinter.
  • System health monitoring (CPU, RAM usage) using psutil.
  • Real-time AI responses powered by the Llama model.
  • Dark mode and sleek design for better UX.

Code Snippet

# Function to get system health data
def get_system_health():
    cpu_usage = psutil.cpu_percent(interval=1)
    ram_usage = psutil.virtual_memory().percent
    return f"CPU Usage: {cpu_usage}%, RAM Usage: {ram_usage}%"

# Function to prompt the AI model
def prompt_model(prompt, model, app):
    response = model(prompt)
    app.update_chat_display(f"AI: {response}")
